## 快速启动

### 1. 启动后端服务

确保你的Express后端服务正在运行：

```bash
cd /home/david/study/jxkc/inventory-system
npm run server
```

后端服务将运行在 `http://localhost:3001`

### 2. 打开微信开发者工具

1. 下载并安装微信开发者工具
2. 选择"导入项目"
3. 项目目录选择：`/home/david/study/jxkc/inventory-system/miniprogram`
4. AppID 可以使用测试号
5. 点击"导入"

### 3. 配置开发工具

在微信开发者工具中：
1. 点击右上角"详情"
2. 在"本地设置"中勾选：
   - ✅ 不校验合法域名、web-view（业务域名）、TLS版本以及HTTPS证书
   - ✅ 启用调试

### 4. 开始使用

- **首页**: 查看库存统计和最近商品
- **商品**: 管理商品信息
- **入库**: 创建入库单据
- **出库**: 创建出库单据

其他页面可通过小程序的页面跳转访问：
- 类别管理：在商品管理页面的"类别"字段中可以跳转
- 客户管理：在出库记录页面选择客户时可以看到
- 供应商管理：在入库记录页面选择供应商时可以看到

## 配置说明

### 修改API地址

如果后端服务不在 `localhost:3001`，修改 `miniprogram/app.js`：

```javascript
globalData: {
  baseUrl: 'http://your-server:port/api',  // 修改为实际地址
  userInfo: null
}
```

### 添加底部导航图标

目前底部导航使用纯文字模式。如需添加图标：

1. 准备图标文件（81×81px PNG格式）
2. 放到 `miniprogram/assets/` 目录
3. 修改 `app.json` 中的 `tabBar` 配置

详细说明请查看 `miniprogram/assets/README.md`

## 常见问题

### 1. 请求失败，网络错误

检查：
- 后端服务是否正常运行
- app.js中的baseUrl配置是否正确
- 微信开发者工具是否勾选了"不校验合法域名"

### 2. 数据显示空白

检查：
- 数据库中是否有数据
- API接口是否正常返回数据
- 查看控制台是否有错误信息

### 3. 页面跳转失败

检查：
- 页面路径是否正确
- app.json中是否注册了该页面
